Projet 5 – Satisfaction client avec SQL

Ce projet consiste à analyser les retours clients pour l’enseigne BestMarket dans le secteur de la grande distribution.

L’objectif est de manipuler une base de données relationnelle pour évaluer la satisfaction client, comprendre les axes d’amélioration et proposer des recommandations pertinentes.
